# Stockout-Risk-Detection-System
Inventory Stockout Risk Detection System using R Programming

## Overview

An R-based inventory analytics project that predicts stockout risks using inventory and sales data.

## Problem Statement
Retail businesses often face stockouts due to poor inventory forecasting and demand mismatch. This leads to revenue loss and poor customer satisfaction.  
This project aims to analyze stockout patterns and detect high-risk products using data analysis techniques.

## Features

- Stockout risk classification
- Days of inventory calculation
- CSV file processing
- Automated reporting
- Visualization using bar charts
- Flow diagram generation

## Technologies

- R Programming
- DiagrammeR
- HTML Widgets

## Key Insights
- Identified high-risk stockout products
- Analyzed demand vs availability gaps
- Found patterns in region-wise stock shortages
- Highlighted critical inventory management issues

---

## Outcomes
- Improved understanding of inventory risks
- Data-driven decision support for stock management
- Interactive dashboards for business insights
